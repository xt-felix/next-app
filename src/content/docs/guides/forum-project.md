## 项目效果说明

这是一个功能完整的论坛项目，包含话题创建、帖子发布、评论回复等核心功能。

### 核心功能

1. **公共头部**：包含帖子搜索和登录功能，存在于每一个页面
2. **首页**：
   - 左侧：总的帖子列表
   - 右侧：创建话题按钮 + 话题列表
3. **帖子详情页**：可以对帖子进行评论，或对别人的评论进行回复
4. **话题帖子列表页**：查看特定话题下的所有帖子，可以创建新帖子

### 功能流程

```
首页 → 点击帖子 → 帖子详情页 → 评论/回复
  │
  └→ 点击 New Topic → 创建话题弹框 → 新话题页面 → 创建帖子
  │
  └→ 点击话题（如 JavaScript）→ 话题帖子列表页 → 创建帖子
```

## 核心页面

| 页面 | 路径 | 功能 |
|------|------|------|
| 首页 | `/` | 展示所有帖子列表和话题列表 |
| 帖子详情页 | `/posts/[id]` | 查看帖子详情、评论和回复 |
| 话题帖子列表页 | `/topics/[slug]` | 查看特定话题下的帖子列表 |

## 技术栈

| 技术 | 说明 |
|------|------|
| **Next.js 15** | 最新版本的 React 全栈框架 |
| **NextUI** | UI 组件库 |
| **TailwindCSS** | CSS 框架 |
| **Prisma** | ORM 工具 |
| **SQLite** | 数据库（可切换为其他数据库） |
| **Auth.js** | 登录认证（NextAuth 最新版本） |

## 项目特点

- 使用 Next.js 15 最新特性
- 完整的用户认证流程
- 话题和帖子的增删改查
- 评论和回复功能
- 搜索功能
- 响应式设计
