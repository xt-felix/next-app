## 什么是 Next.js?

Next.js 是一个基于 React 全栈框架，由 Vercel 开发和维护，那么它有什么优势呢?

<CardGrid>
  <Card title="SSR" icon="rocket">
    服务端渲染，可以提高页面加载速度（现在会 SSR 的人才并不多，可以提升你的竞争力）
  </Card>
  <Card title="SSG" icon="document">
    静态站点生成，可以生成静态页面，类似于 Vitepress / Astro 等静态站点生成工具
  </Card>
  <Card title="SEO" icon="magnifier">
    Next.js 提供了 SEO 优化，让你的网站更容易被搜索引擎收录
  </Card>
  <Card title="AI" icon="star">
    Vercel 提供了 AI SDK 可以跟 Next.js 轻松结合，让你可以轻松实现 AI 应用
  </Card>
</CardGrid>

## 前置知识

在学习 Next.js 之前，你需要掌握以下知识：

- HTML / CSS / JavaScript 基础
- React 基础（组件、状态、生命周期等）
- Node.js 基础
